Java Logic Utilities with Spring Boot

Palindrome Check

A palindrome is a word or phrase that reads the same backward as forward. Palindromes ignore spaces, punctuation, and capitalization during evaluation.

Example: 'madam'.

Factorial Calculation

The factorial of a number is the product of all positive integers up to that number. Factorial is defined only for non-negative integers.

Example: 4! = 4 × 3 × 2 × 1 = 24.

URL Encoding

URL encoding replaces certain characters in a URL with their ASCII equivalents. It converts spaces to '%20'.

Example: 'Hello World' → 'Hello%20World'.

Substring Check

A substring is a sequence of characters within a string. Substrings can vary in length and are derived from the original string.

Example: In "hello," "ell" is a substring.

Matrix Transpose

The transpose of a matrix swaps its rows and columns. Please input the matrix in the following format:

1 2 3
4 5 6
7 8 9

Unique Characters Check

Checks if all characters in a string are unique. Each character should appear only once for the string to qualify as unique.

Example: "apple" → false, "kite" → true.
